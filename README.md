Bienvenido a **Agent Clinic**, una aplicación moderna desarrollada con [Next.js](https://nextjs.org/). 

## 🚀 Características Principales

- **Interfaz Moderna:** Construida aprovechando las últimas características de Next.js (App Router).
- **Desarrollo Guiado por Especificaciones (SDD):** El comportamiento de la aplicación está rigurosamente definido y validado a través de archivos de pruebas (`.spec`).
- **Optimización Automática:** Uso de `next/font` con la tipografía Geist para una carga rápida y optimizada.

## 🛠️ Tecnologías Utilizadas

- **Framework:** [Next.js](https://nextjs.org/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/) / JavaScript
- **Estilos:** Tailwind CSS
- **Entorno:** Node.js
